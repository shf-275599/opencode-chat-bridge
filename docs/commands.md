# 命令手册 (Commands)

`opencode-lark` 支持多种交互式命令，直接在 IM 聊天窗口中输入即可触发管理操作。

## 1. 会话管理命令

| 命令 | 说明 | 示例 |
|---|---|---|
| `/new` | 强行开启一个新的 opencode 会话，并绑定到当前聊天 | `/new` |
| `/sessions` | 列出最近的 TUI 会话及其绑定状态 | `/sessions` |
| `/connect` | 手动将当前聊天绑定到指定的 Session ID | `/connect ses_abc123` |
| `/abort` | 中止 Agent 当前正在运行的任务（如卡住的工具调用） | `/abort` |

## 2. 状态与管理

| 命令 | 说明 | 示例 |
|---|---|---|
| `/help` | 显示帮助菜单 | `/help` |
| `/share` | 生成并返回当前会话的内网/公网分享链接 | `/share` |
| `/compact` | 强制执行上下文压缩，清理冗长的对话历史 | `/compact` |

## 3. 可靠性管理 (仅限管理员)

| 命令 | 说明 | 示例 |
|---|---|---|
| `/cron` | 查看当前正在运行的定时任务列表 | `/cron` |
| `/heartbeat` | 查看心跳探针状态（成功/失败计数） | `/heartbeat` |

## 4. 平台差异

- **飞书**: 以上命令均返回美观的 **交互式卡片**，您可以直接点击卡片上的按钮进行操作。
- **QQ / Telegram**: 命令以 **纯文本格式** 返回结果或引导语。
