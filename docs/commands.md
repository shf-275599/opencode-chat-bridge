# 命令手册 (Commands)

`opencode-im-bridge` 支持多种交互式命令，可在 IM 聊天窗口或 TUI 中直接使用。

## TUI 快捷键 ↔ IM 命令对照

| TUI 快捷键 | IM 命令 | 功能 |
|------------|---------|------|
| `Ctrl+N` | `/new` | 新建会话 |
| `Ctrl+L` | `/sessions` | 列出会话 |
| `Ctrl+Shift+C` | `/connect` | 连接会话 |
| `Ctrl+Shift+X` | `/abort` | 中止任务 |
| `Ctrl+M` | `/compact` | 压缩上下文 |
| `Ctrl+S` | `/share` | 分享会话 |
| `Ctrl+H` | `/help` | 帮助菜单 |

---

## IM 命令详解

### 1. 会话管理命令

| 命令 | 说明 | 示例 |
|---|---|---|
| `/new` | 强行开启一个新的 opencode 会话，并绑定到当前聊天 | `/new` |
| `/sessions` | 列出最近的 TUI 会话及其绑定状态 | `/sessions` |
| `/connect {session_id}` | 手动将当前聊天绑定到指定的 Session ID | `/connect ses_abc123` |
| `/abort` | 中止 Agent 当前正在运行的任务（如卡住的工具调用） | `/abort` |

### 2. 状态与管理

| 命令 | 说明 | 示例 |
|---|---|---|
| `/help` 或 `/` | 显示帮助菜单 | `/help` |
| `/share` | 生成并返回当前会话的内网/公网分享链接 | `/share` |
| `/compact` | 强制执行上下文压缩，清理冗长的对话历史 | `/compact` |

### 3. 可靠性管理 (仅限管理员)

| 命令 | 说明 | 示例 |
|---|---|---|
| `/cron` | 查看当前正在运行的定时任务列表 | `/cron` |
| `/heartbeat` | 查看心跳探针状态（成功/失败计数） | `/heartbeat` |

### 4. 问答交互命令

| 命令 | 说明 | 示例 |
|---|---|---|
| `/ask {问题}` | 以问答卡片形式提问，AI 回复后提供预设选项 | `/ask 帮我选哪个方案` |
| `/poll {选项}` | 创建多选投票卡片 | `/poll 方案A 方案B 方案C` |

---

## 平台差异

| 平台 | 命令返回形式 |
|------|-------------|
| **飞书** | 交互式卡片，支持点击按钮操作 |
| **QQ** | 纯文本格式 |
| **Telegram** | 纯文本格式 + Inline 按钮 |
| **Discord** | Embed 消息 |

## Q&A 卡片交互

当 AI 需要向你提问或征求确认时，会发送 **问答卡片**：

```
┌─────────────────────────────────┐
│ 🤖 AI 助手 需要你的回复          │
├─────────────────────────────────┤
│  请选择一个选项：               │
│                                 │
│  [✅ 是的，继续]  [❌ 取消]      │
│                                 │
│  或直接回复你的答案              │
└─────────────────────────────────┘
```

**交互方式**：
1. 点击卡片按钮直接选择
2. 在聊天框输入你的回复
3. 发送 `/skip` 跳过当前问答
