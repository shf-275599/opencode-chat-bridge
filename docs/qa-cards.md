# Q&A 卡片交互文档

opencode-im-bridge 支持通过飞书交互式卡片实现问答交互，用户可以直接在聊天窗口中选择预设答案或输入自定义回复。

## 功能特性

- **预设选项**: AI 可发送带按钮的卡片，用户点击即选
- **自定义回复**: 用户也可直接在聊天框输入答案
- **多轮对话**: 支持连续的问答流程
- **超时处理**: 可设置问答超时时间和默认值

## 卡片类型

### 1. 确认卡片 (Confirm Card)

用于征求用户确认，如权限申请、危险操作等。

```
┌────────────────────────────────────┐
│ ⚠️ 操作确认                         │
├────────────────────────────────────┤
│ 是否确认执行以下操作？               │
│                                    │
│ • 删除文件: /tmp/test.log           │
│                                    │
│ [✅ 确认执行]    [❌ 取消操作]       │
└────────────────────────────────────┘
```

### 2. 选择卡片 (Choice Card)

用于让用户从多个选项中选择。

```
┌────────────────────────────────────┐
│ 🤔 请选择方案                        │
├────────────────────────────────────┤
│ 请问使用哪个部署方案？               │
│                                    │
│ [🚀 方案A: Docker]                  │
│ [⚡ 方案B: K8s]                    │
│ [📦 方案C: VM]                     │
│ [✏️ 其他方案]                       │
└────────────────────────────────────┘
```

### 3. 输入卡片 (Input Card)

用于需要用户提供文本输入的场景。

```
┌────────────────────────────────────┐
│ 💬 请输入                          │
├────────────────────────────────────┤
│ 请输入文件名（不含特殊字符）：        │
│                                    │
│ ┌────────────────────────────┐     │
│ │ 请输入...                  │     │
│ └────────────────────────────┘     │
│                                    │
│ [提交]           [跳过]            │
└────────────────────────────────────┘
```

## 使用方式

### 通过 Agent 触发

Agent 可以通过发送特殊格式的消息来触发问答卡片：

```markdown
@[card:confirm]
动作: 删除文件
文件: /tmp/test.log
超时: 60秒
选项:
  - 确认执行
  - 取消操作
@[end]
```

### 通过 IM 命令触发

```
/ask {问题}    # 以问答卡片形式提问
/poll {选项1} {选项2} {选项3}  # 创建多选投票
```

## 响应处理

### 用户点击按钮

用户点击按钮后，系统自动解析并发送对应的选择给 opencode。

### 用户输入文本

用户也可以直接在聊天框输入，优先级高于按钮选择。

### 超时处理

如果用户超时未响应，可以配置：
- 默认选择
- 跳过并继续
- 取消操作

## 平台支持

| 平台 | 支持度 | 说明 |
|------|--------|------|
| **飞书** | ✅ 完全支持 | CardKit 交互式卡片 |
| **QQ** | ⚠️ 部分支持 | Inline 键盘按钮 |
| **Telegram** | ✅ 完全支持 | Inline 键盘按钮 |
| **Discord** | ✅ 完全支持 | Embed + 按钮 |

## 配置选项

在 `opencode-im-bridge.jsonc` 中配置：

```jsonc
{
  "qa": {
    "enabled": true,
    "defaultTimeout": 300,        // 默认超时时间（秒）
    "allowTextInput": true,      // 允许文本输入
    "enableNotifications": true   // 超时提醒
  }
}
```
