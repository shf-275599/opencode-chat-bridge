### 快捷键对照表 (TUI ↔ IM)

| TUI 快捷键 | IM 命令 | 功能 |
|------------|---------|------|
| `Ctrl+N` | `/new` | 新建会话 |
| `Ctrl+L` | `/sessions` | 列出会话 |
| `Ctrl+Shift+C` | `/connect` | 连接会话 |
| `Ctrl+Shift+X` | `/abort` | 中止任务 |
| `Ctrl+M` | `/compact` | 压缩上下文 |
| `Ctrl+S` | `/share` | 分享会话 |
| `Ctrl+H` | `/help` | 帮助菜单 |
